# leandrocassius.github.io

Personal portfolio site — terminal-themed, dark aesthetic.

## Setup

1. Create a new repository on GitHub named **leandrocassius.github.io** (must match your username).
2. Push this folder to it:

```bash
cd leandrocassius.github.io
git init
git add .
git commit -m "Initial portfolio site"
git branch -M main
git remote add origin https://github.com/leandrocassius/leandrocassius.github.io.git
git push -u origin main
```

3. GitHub Pages will auto-deploy. The site will be live at **https://leandrocassius.github.io** within a few minutes.
